Fundamentals of CSS

TL;DR

Background color, text color, font family, font size, image height/width

Overview

The most common styling methods in websites: Background color, text color, font family, font size, image height/width etc.

Links

Learn more about CSS: W3Schools .
Mozilla Developer Reference: Mozilla .
Google Fonts API: Google Fonts .

Background color

The background-color CSS property sets the background color of an element.


You must assign a color to the background-color in order to use it.


You can use a Keyword value, Hexadecimal value, RGB value or HSL value to assign a color to it.


Credits: Mozilla

Text color


The color CSS property sets the foreground color value of an element's text and text decorations


You must assign a color to the color in order to use it.


You can use a Keyword value (e.g. ), Hexadecimal value, RGB value or HSL value to assign a color to it.


Credits: Mozilla

Font Family

The font-family CSS property sets the font that a element uses.


You can use a font on your computer, a .ttf(True Type Font) file or from an API(Application Programming Interface).


Font: Raleway

Font size

The font-size CSS property sets the size of the font.


You must assign a value to font-size in order change the font size.


Credits: Mozilla

Image Size

The width and height CSS property sets the size of a image.


You must assign a value to both width and height in order change the image size.


Reference: w3schools
