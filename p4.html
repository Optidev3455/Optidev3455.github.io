Extension on CSS

TL;DR

hover, transition, media query, keyframe

Overview

The essential CSS properties for web development which almost every website uses, these properties will make your website more clean.

Links

Learn more about CSS: W3Schools .
Mozilla Developer Reference: Mozilla .

Hover

The :hover pseudo-class matches when the user interacts with an element with a pointing device, but does not necessarily activate it.


It is generally triggered when the user hovers over an element with the cursor (mouse pointer).


Credits: Mozilla

Transition


The transition CSS property enable you to define the transition between two states of an element.


You must assign a time value to the transition property in order to use it.


Credits: Mozilla

Media Query

The @media or media queries allow you to apply CSS styles depending on a device's media type (such as print vs. screen)


You can use other features or characteristics such as screen resolution or orientation, aspect ratio, browser viewport width or height, user preferences such as preferring reduced motion, data usage, or transparency.


This is the main CSS property to create responsive web design.


The following image will change size based on your device display width.


Credits: Mozilla

Keyframes

The @keyframes CSS at-rule controls the intermediate steps in a CSS animation sequence by defining styles for keyframes (or waypoints) along the animation sequence.

You must assign a name to the name of your keyframe.


Reference: w3schools
