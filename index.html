Tools used during meetings

IDEs and Hacking tools

VS Code

Light Weight IDE

Visual Studio Code is a powerful all purpose IDE developed by Microsoft. This IDE supports many languages and is light weight, making it a very useful tool for programmers.

IntelliJ IDEA

Full blown IDE

IntelliJ IDEA is the perfect IDE for java developers. With it's better code completion and more frameworks makes it so popular and many consider it as the best IDE for java.

Dev C++

Light Weight IDE

A C compiler developed by Embarcadero. It has a fast compilation speed. But with lack of code assistance, it is not very suitable for beginners learning C or C++.

Wireshark

Packet Analyzer

A network tool developed by The Wireshark Team. It is great for network troubleshooting, analysis, software and communications protocol development.

Nmap

Network Scanner

Developed by Gordon Lyon. Nmap is used to discover hosts and services on a computer network by sending packets and analyzing the responses. As well as OS detection.

hPing3

Network Tool

Developed by Salvatore Sanfilippo. hPing3 is used for TCP/IP analyzing. Many cybersecurity experts uses it for security auditing and testing of firewalls and networks.

Burp Suite

Penetration Testing

Developed by PortSwigger. It is used for security testing of web applications. It lets you combine AMTs with SOTA automation, to make your work faster.
