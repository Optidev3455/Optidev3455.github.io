Installing Required Software

TL;DR

We will be installing Visual Studio Code with the extension: Live Preview for HTML development.

Overview

Visual Studio Code (VS Code) and the Live Preview extension provide a powerful and efficient environment for web development. By following the steps below, you can quickly set up VS Code with the extension Live Preview on your system.

Links

VS Code Download: Official Download Page .
Live Preview Download: Visual Studio Marketplace .

Installing Visual Studio Code

We will be using Visual Studio Code (VS Code) for web development throughout the entire club meeting.


To download VS Code, visit the official website and then select the download option for your OS.


After downloading the installer, run the installer and follow the instructions listed on the installer wizard.

Installing Live Preview Extension

We will be using the Live Preview extension by Microsoft for real time HTML updates to boost productivity.


To download the extension, either visit the Visual Studio Marketplace and then run the downloaded file or install it directly within VS Code through its built-in marketplace.


To download it directly in VS Code, open the Extensions view on the left sidebar (Shortcut: Ctrl+Shift+X) and search for Live Preview in the search bar.


After that, click install and wait for it to be installed on your system.

Using the Live Preview Extension

To use the extension we just downloaded, go to the top right corner of VS Code and click the "Show Preview" button (The grid with a magnifying glass).


By clicking the button, the extension will automatically open your local ip with the default port of 3000 to a panel next to your editor with a real time update of what you've written in the editor, which is very helpful for debugging.


You can also access it via your browser by typing either 127.0.0.1:3000 or localhost:3000 into the url bar to load your website there, where you can have a better way to debug your website (e.g. Element inspection, responsive preview).


To close/stop the extension on your keyboard, use the key combination (Ctrl+Shift+P) to open up the Command Palette and search for Live Preview and locate the option for "Live Preview: Stop Server" to terminate the process.
