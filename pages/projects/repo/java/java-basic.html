Basic Java Syntax


Before we start, we'll create a java file called Main.java To write the codes here.

Comments

The // symbol is used to create a single-line comment in Java and /* */ is used to make a multi-line comment.
Comments will be ignored by the compiler during compilation, but it is a great way of making notes or documenting your code.

Print Line

Java's method of printing a line is quite interesting. All code in Java MUST be inside a class. To start, we'll create a class with the name Main because the file name we've created is called Main.java and it is required due to it being a public method.
Initializing the class
public class Main {
// This must be the same as your file name.
/* The above is a single-line comment and
this is a multi-line comment. */
}


After that we'll create another line using the public static void method. And it will be called main(). The main method is required as it tells the compiler a.k.a. JVM (Java Virtual Machine) that this is the entry point of your code. Then inside the brackets we'll add String[] args.
Initializing the main method
public class Main {
    public static void main(String[] args) {

    }
}


Now I'll explain what we've just done.

public is an access modifier in Java and everything in it (e.g. classes, variables) can be accessed throughout the entire file a.k.a global value. We're using it together with the class because throughout the entire class we'll be only using the public method.

static is a keywork used for memory management which saves memory(RAM) during compilation. static can also be used as a variable.

void is also a keyword used to specify a method to not return anything. Becuase the main method does not return anything so we'll use the void keyword to tell the JVM to terminate the code after it has run everything inside the main method.

String[] args is a string array which arguments are passed here. It is a command-line argument for java.lang.String class. The array is only the square brackets [] so you can call args whatever you want.

To print a line in Java, we'll type System.out.println followed by a bracket.

System.out is the out put stream in Java which is usually followed by println(Print Line) to print something.
All statements in Java MUST end with a semi colon ; to tell the JVM that the statement terminates here.
Print line in Java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
        /* You can use the print line statement 
        without double quotes to print numbers or do calculations */
        System.out.println(50 + 50);
    }
}



Basic Data Types



Characters Based:


String stores a string value which is a sequence of characters.
Size in memory if empty: 40 bytes
example:
String Data Type
// Assigning a String variable
String i = "Hello World";
// Variable i now stores a String value of Hello World


char stores a single character which can be in ASCII format.
Size in memory if empty: 2 bytes
example:
char Data Type
// Assigning a char variable
char i = 'A';
// Variable i now stores the character A


Numeric Based:


int, byte, short, long stores an integer value and CANNOT store any decimal values.
example:
int, byte, short and long Data Types
// Assigning a int, byte, short and long variable
int i = 50;
byte x = 50;
short y = 50;
long z = 50;
// All Variable now stores the numeric value of 50
/* 
int can store numbers from -2,147,483,647 to 2,147,483,647
byte can store numbers from -128 to 127
short can store numbers from -32,768 to 32,767
long can store number from -9,223,372,036,854,775,808 
to 9,223,372,036,854,775,807
*/


float, double stores a floating point value and CAN store decimal values.
example:

float and double Data Type
// Assigning float and double variable
float i = 50.5;
double x = 50.5;
// All Variable now stores the floating point value of 50.5


Boolean Based:

boolean stores a true or false value.
example:

Bool Data Type
boolean a = true;
boolean b = false;
boolean c = 50;
boolean d = 50;
/* 
Variable a now stores a boolean value of 1 or true
Variable b now stores a boolean value of 0 or false
Variable c now stores a boolean value of 1 or true
Variable d now stores a boolean value of 0 or false
Because boolean value is only 1 or 0, values higher than 1
will be stored as 1. So variable e now stores a 
boolean value of 1 or true 
*/


Operators

+ Addition
example: 1 + 1 = 2

- Subtraction
example: 2 - 1 = 1

* Multiplication
example: 2 * 2 = 4

% Modulus
example: 4 % 3 = 1 (The remainder of 4 / 3)

/ Division
example: 50 // 10 = 5

(variable)++ Increment
example: Increase the value of the variable by 1.

(variable)-- Decrement
example: Decrease the value of the variable by 1.

All Operators can be use with a = assignment operator to directly assign a value to the corresponding variable.
